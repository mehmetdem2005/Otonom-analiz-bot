3. DASHBOARD PENCERELERI

   a) Genel Bakış
      - 4 istatistik kartı (Çalışan Ajanlar, İşlenen İstekler, Hatalar, API Çağrıları)
      - Son loglar paneli
      - Real-time güncelleme

   b) Ajan İzleme
      - 10 ajanın izleme pencereleri (grid layout)
      - Her ajan için: Durum, İşlem Sayısı, Hata Sayısı, Son Çalıştırma
      - Renkli status göstergeleri (çalışıyor/uyarı/hata)
      - Son çıktı preview

   c) Konuşmalar
      - Her ajanın konuşmasını gösteren log pencereleri
      - Zaman damgası ile çıktı
      - Renkli level göstericileri (INFO/WARN/ERROR/SUCCESS)

   d) Ajan İletişimi
      - Ajanlar arası mesajları gösteren dedikasyon penceresi
      - Format: [A0 → A5] mesaj içeriği

   e) Tüm Loglar
      - Sistem genelindeki tüm loglar
      - Tek bir scroll-able pencerenin içinde
      - Max 500 log tutuluyor

   f) Hata Logları
      - Sadece ERROR seviyesi loglar
      - Reddedilmiş hatalar vurgulu
      - Hata sayacı güncelleniyor

   g) Database Yönetimi
      - 4 konu seçim: Hafıza, Uyarılar, Tweetler, Arşiv
      - JSON viewer (formatted)
      - Veritabanı içeriğine erişim

   h) AI Gelişimi Analizi
      - Öz-İyileştirme Oranı (%) - ilerleme barı
      - API Bağımlılığı Azalması (%) - ilerleme barı
      - Sistem Parametreleri (Toplam, Eğitli, Hassasiyet, Adaptasyon)
      - Yetenekler listesi (✅ 6 özellik)

4. WEB API ENDPOİNTLERİ (/web_arayuzu.py'ye eklendi)

   GET /dashboard
   - Dashboard HTML'i sunuyor
   
   GET /api/agents
   - 10 ajanın adlarını JSON'da döndürüyor
   
   GET /api/analytics
   - runningAgents (10)
   - processedRequests (sonuç sayısı)
   - errorCount (hata sayısı)
   - apiCalls (API istem sayısı)
   - selfImprovementRate (%) - öz-düzeltme
   - apiDependencyReduction (%) - yerel vs API işlem oranı
   - totalParameters (10 × 512k)
   - trainedParameters (bağımlılık azalması oranında)
   
   GET /api/database/{filename}
   - hafiza.json, alerts.json, tweets.json, archive.json, comment-profiles.json
   - Güvenlik: izin verilen dosyalar listesi
   
   GET /api/logs/agent/{agent_name}
   - Belirli ajanın loglarını filtreleyip döndürüyor
   
   GET /api/disk-usage
   - Toplam/Kullanılan/Boş disk (GB)
   - Yüzde
   
   WS /ws/agent-status
   - Real-time ajan durumu
   - 2 saniyede bir güncelleme
